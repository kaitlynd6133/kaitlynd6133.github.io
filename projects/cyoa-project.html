Cyao Project

Context

In my create your own adventure project, I used my knowledge of github basics such as ```git add .```, ```git commit -m``` and ```git status``` to help create my project. I used merging and especially in using my skills on linking things in the IDE. For this project I decided to go with the topic zombie invasion and create a story with that topic.

Content

In this project I used Github and CS50 IDE. When making the project, me and my partner were first required to make a drawing plan. After that we needed to create a repo including a Readme file. In the ReadME we had to add a link to our drawing plan and links to our github profiles. Then we were required to add a minimum of 10 files and add a sub-directory. We added many files linking together. We meet these requirments by working together and keeping in touch through insta.
